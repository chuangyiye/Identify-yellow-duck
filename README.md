## 1.项目简介
## Identify-yellow-duck是一个基于yolov5实现识别物体的项目
## 2.功能简述
### 此版本功能包含了识别物体和项目的一下特色
## 识别物体:
- 识别照片中的小黄鸭
- 通过摄像头识别物体小黄鸭
